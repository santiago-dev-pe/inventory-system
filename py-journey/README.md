# Python Journey

This repository tracks my progress in Python development, ranging from basic library exploration to the creation of functional applications.

## Project Structure

### 01 Practice
This directory contains focused exercises and library testing. It is organized into subcategories to maintain a clear learning path:
- Basics: Fundamental syntax and core logic.
- Library Exploration: Hands-on practice with specific packages like logging, pathlib, and others.

### py-apps
This folder hosts small, functional applications. Unlike the practice scripts, these projects solve specific problems and usually consist of multiple interconnected files, demonstrating higher complexity and software design.

## Technical Environment
- Language: Python 3.x
- Tools: Virtual environments and pip for dependency management.
